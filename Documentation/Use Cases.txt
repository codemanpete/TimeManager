## Use Cases
**1. Create Account**
 a. User opens GUI.
 b. User selects 'create account'.
 c. User provides required fields (username, password, name, last name).
 d. User selects 'OK'.

**2. Update Account**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User selects 'account' menu and selects 'edit account'.
 d. User enters new fields (password, name, last name).
 e. User selects 'OK'.

**3. Create Appointment**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User selects 'appointment' menu and selects 'new appointment'.
 d. User provides required fields (title, start time, duration, notes, other users).
 e. User selects 'OK'.

**4. Edit Appointment**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User browses the Calendar View and selects an Appointment.
 d. User provides updated fields.
 e. User selects 'OK'.

**5. Delete Appointment**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User browses the Calendar View and selects an Appointment.
 d. User selects 'delete' and confirms.

**6. View Calendar by Month**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User selects 'Month View'.

**7. View Calendar by Week**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User selects 'Week View'.

**8.  Search for User**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User selects 'users' menu and selects 'browse'.
 d. User enters search username in text field.
 e. User selects a User from list.

**8a. Add User to Contact List**
 f. User selects 'add to contacts'.

**8b. Add User to Team**
 f. User selects 'add to team'.
 g. User selects team.
 h. User selects 'OK'.

**9. View Contact List**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User selects 'contacts' menu and selects 'view contacts'.

**9a. Remove User from Contact List**
 d. User selects user.
 e. User selects 'remove from contacts'.

**10. Create Team**
 a. User opens GUI.
 b. User selects 'login' and provides credentials.
 c. User selects 'team' menu and selects 'create new team'
 d. User enters required fields (team name).
 e. User selects 'OK'.
